# 📌 Image Sharing Platform

Aplikasi web berbasis PHP Native yang terinspirasi dari konsep Pinterest. Pengguna dapat membuat akun, login, dan mengelola postingan gambar melalui fitur CRUD (Create, Read, Update, Delete).

## ✨ Fitur

### Authentication

* Register akun
* Login pengguna
* Logout

### Post Management

* Menampilkan daftar postingan
* Melihat detail postingan
* Menambahkan postingan baru
* Mengedit postingan
* Menghapus postingan
* Upload gambar

## 🛠️ Tech Stack

* PHP Native
* MySQL
* HTML
* CSS
* JavaScript
* Bootstrap

## 📚 Konsep yang Diterapkan

* CRUD (Create, Read, Update, Delete)
* Authentication & Session
* Form Validation
* File Upload
* Database Relational
* Object-Oriented Programming (OOP)

## 🗂️ Struktur Fitur

### User

* Registrasi akun
* Login dan Logout
* Membuat postingan
* Mengedit postingan sendiri
* Menghapus postingan sendiri
* Melihat seluruh postingan

## 🚀 Instalasi

### Clone Repository

```bash
git clone https://github.com/username/project-name.git
```

### Pindahkan ke Web Server

Simpan project pada folder:

**XAMPP**

```text
htdocs/project-name
```

**Laragon**

```text
www/project-name
```

### Buat Database

Buat database baru melalui phpMyAdmin, misalnya:

```sql
CREATE DATABASE image_sharing_platform;
```

### Import Database

Import file SQL yang tersedia pada folder project.

### Konfigurasi Database

Sesuaikan file konfigurasi database:

```php
define('DBHOST', 'localhost');
define('DBNAME', 'image_sharing_platform');
define('DBUSER', 'root');
define('DBPASS', '');
```

### Jalankan Aplikasi

Buka browser dan akses:

```text
http://localhost/php-images-website
```
